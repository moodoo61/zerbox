# مقترح مشروع المنصة الترفيهية والخدمية

## 🎯 الهدف من المشروع
إنشاء نظام ويب يعمل على سيرفر Ubuntu متصل بالشبكة المحلية، يسمح بعرض خدمات موجودة على الخادم وإدارتها عبر لوحة تحكم مخصصة.

## 🛠️ مقترح التقنيات والهيكل
| العنصر | التقنية |
|---|---|
| Backend | FastAPI |
| Frontend | React + Tailwind CSS |
| قاعدة البيانات | SQLite |
| الحماية | Password Session |
| النظام | Linux Ubuntu |
