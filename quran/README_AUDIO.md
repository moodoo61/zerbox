# تنزيل الملفات الصوتية - Audio Download Guide

## نظرة عامة

هذا السكريبت يقوم بتنزيل جميع الملفات الصوتية للقرآن الكريم وتخزينها محلياً بدلاً من الاعتماد على الروابط الخارجية.

## المميزات

- ✅ تنزيل جميع الملفات الصوتية (158 قارئ × 114 سورة = ~18,000 ملف)
- ✅ حفظ الملفات محلياً في `data/audio/`
- ✅ تحديث روابط JSON لاستخدام الملفات المحلية
- ✅ الحفاظ على الروابط الأصلية كنسخة احتياطية
- ✅ تجاوز الملفات المتنزلة مسبقاً
- ✅ معالجة الأخطاء وإعادة المحاولة

## الاستخدام

### 1. تنزيل جميع الملفات الصوتية

```bash
npm run downloadAudio
```

### 2. تنزيل سورة محددة

يمكنك تعديل السكريبت لتنزيل سورة واحدة فقط:

```javascript
// في scripts/downloadAudio.mjs
await processSurahAudio(1); // سورة الفاتحة فقط
```

## ملاحظات مهمة

### ⚠️ حجم الملفات

- **حجم تقريبي**: ~15-20 جيجابايت للكامل
- **وقت التنزيل**: عدة ساعات حسب سرعة الإنترنت
- **مساحة التخزين**: تأكد من وجود مساحة كافية

### 🔄 التنزيل التدريجي

يمكنك إيقاف التنزيل والعودة لاحقاً:
- السكريبت يتحقق من الملفات الموجودة مسبقاً
- يواصل التنزيل من حيث توقف

### 📊 التقدم

السكريبت يعرض:
- عدد الملفات المتنزلة
- عدد الملفات المسبق التنزيل
- عدد الملفات الفاشلة

### 🔧 استكشاف الأخطاء

إذا فشل تنزيل ملفات:
- الملفات الفاشلة تحتفظ بالرابط الأصلي
- يمكن إعادة تشغيل السكريبت للتنزيل مرة أخرى
- تحقق من الاتصال بالإنترنت

## تحديث الروابط

بعد التنزيل، الروابط في JSON تصبح:
```json
{
  "id": 1,
  "reciter": { "ar": "أحمد الحواشي" },
  "link": "/data/audio/surah_1/reciter_1_001.mp3",
  "originalLink": "https://server11.mp3quran.net/hawashi/001.mp3"
}
```

الخادم يخدم هذه الملفات تلقائياً من `/data/audio/`.
