# دليل التشغيل التلقائي - Quran Data

## طرق التشغيل

### الطريقة 1: استخدام ملف Batch (الأسهل) ⭐
1. انقر نقراً مزدوجاً على `start.bat`
2. سيقوم الملف تلقائياً بـ:
   - التحقق من تثبيت Node.js
   - تشغيل الخادم
   - فتح المتصفح

### الطريقة 2: استخدام PowerShell Script
1. انقر بزر الماوس الأيمن على `start.ps1`
2. اختر "Run with PowerShell"
3. أو شغله من PowerShell:
   ```powershell
   .\start.ps1
   ```

### الطريقة 3: استخدام npm
```bash
npm run launch
```

### الطريقة 4: إنشاء اختصار على سطح المكتب
1. شغّل `إنشاء_اختصار.bat`
2. سيتم إنشاء اختصار على سطح المكتب
3. انقر نقراً مزدوجاً على الاختصار لتشغيل التطبيق

## الملفات المتوفرة

- `start.bat` - ملف Batch لتشغيل التطبيق (Windows)
- `start.ps1` - سكريبت PowerShell
- `launch.js` - سكريبت Node.js للتشغيل التلقائي
- `إنشاء_اختصار.bat` - إنشاء اختصار على سطح المكتب

## ملاحظات

- تأكد من تثبيت Node.js قبل التشغيل
- الخادم يعمل على المنفذ 5000
- المتصفح يفتح تلقائياً على `http://localhost:5000`
- لإيقاف الخادم، أغلق نافذة الخادم أو اضغط Ctrl+C

## استكشاف الأخطاء

### الخادم لا يبدأ
- تحقق من تثبيت Node.js: `node --version`
- تحقق من تثبيت التبعيات: `npm install`
- تأكد من أن المنفذ 5000 غير مستخدم

### المتصفح لا يفتح تلقائياً
- افتح المتصفح يدوياً على: `http://localhost:5000`
