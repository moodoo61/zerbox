# واجهة القرآن الكريم - Quran Data Frontend

واجهة أمامية حديثة وتفاعلية لعرض وقراءة القرآن الكريم باستخدام API الخادم.

## المميزات

### 📖 عرض السور
- عرض جميع سور القرآن الكريم في شبكة جميلة
- ترتيب السور (رقمي، أبجدي، حسب عدد الآيات)
- معلومات كل سورة (عدد الآيات، الكلمات، الحروف)

### 📜 عرض الآيات
- عرض جميع آيات السورة
- عرض النص العربي والإنجليزي
- معلومات كل آية (الجزء، الصفحة)

### 🎧 مشغل الصوتيات
- تشغيل التسجيلات الصوتية للقرآن
- مشغل صوتي متقدم مع شريط التقدم
- دعم عدة قراء

### 📄 صفحات القرآن
- تصفح صفحات القرآن الكريم
- عرض صور الصفحات
- التنقل بين الصفحات

### 🕋 آيات السجدة
- عرض جميع آيات السجدة
- تمييز السجدة المستحبة والواجبة
- الانتقال مباشرة إلى السورة

### 🔍 البحث
- البحث في أسماء السور
- البحث بالعربية والإنجليزية

## الاستخدام

1. تأكد من أن الخادم يعمل على `http://localhost:5000`
2. افتح المتصفح على `http://localhost:5000`
3. استمتع بتصفح القرآن الكريم!

## التقنيات المستخدمة

- HTML5
- CSS3 (مع تصميم حديث ومتجاوب)
- JavaScript (Vanilla JS)
- Google Fonts (Amiri, Cairo)
- API RESTful

## التصميم

- تصميم حديث وجميل
- متجاوب مع جميع الأجهزة
- ألوان هادئة ومريحة للعين
- خطوط عربية جميلة (Amiri للقرآن، Cairo للواجهة)

## الملفات

- `index.html` - الصفحة الرئيسية
- `style.css` - ملف التصميم
- `app.js` - منطق التطبيق
