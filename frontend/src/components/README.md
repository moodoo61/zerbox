# مكونات واجهة المستخدم

هذا الدليل يحتوي على جميع مكونات واجهة المستخدم المنظمة في هيكل مودولي لسهولة التطوير والصيانة.

## 🗂️ هيكل المجلدات

```
components/
├── layout/          # مكونات التخطيط العام
│   └── Navbar.js    # شريط التنقل الرئيسي
├── hero/            # مكونات القسم الرئيسي  
│   └── HeroSection.js # القسم التقديمي الرئيسي
├── services/        # مكونات الخدمات والتطبيقات
│   ├── ServiceCard.js     # بطاقة الخدمة الواحدة
│   └── ServicesSection.js # قسم عرض جميع الخدمات
└── index.js         # ملف الصادرات المركزي
```

## 🎨 الأنماط المشتركة

```
styles/
├── animations.js    # الحركات والتأثيرات البصرية
└── index.js        # ملف الصادرات للأنماط
```

## 📱 المكونات

### Navbar
شريط التنقل المتجاوب مع:
- تصميم شفاف مع تأثير blur
- قائمة جانبية للهواتف المحمولة
- أزرار التنقل المحسنة

### HeroSection  
القسم الرئيسي مع:
- خلفية ديناميكية (صورة أو لون)
- عرض سريع للخدمات
- أزرار الإجراءات الرئيسية
- تأثيرات بصرية متقدمة

### ServiceCard
بطاقة الخدمة المحسنة مع:
- تصميم متدرج وجذاب
- تأثيرات hover متقدمة
- شارات ومعلومات إضافية
- أزرار عمل محسنة

### ServicesSection
قسم عرض الخدمات مع:
- رأس القسم مع إحصائيات
- شبكة متجاوبة للبطاقات
- حالة فارغة جذابة
- زر عائم للتنقل

## 🚀 الاستخدام

```javascript
import { HomePage, Navbar, HeroSection } from '../components';

// أو استيراد مكون واحد
import Navbar from '../components/layout/Navbar';
```
